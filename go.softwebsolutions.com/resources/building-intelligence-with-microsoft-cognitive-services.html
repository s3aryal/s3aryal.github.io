About Event

Microsoft Cognitive Services enables you to build apps with powerful algorithms just by using a few lines of code. It helps you to easily and quickly inject intelligence in your apps by adding powerful APIs for image recognition and analysis; facial recognition, speech recognition, text-to-speech capabilities, and more.

At this event, the attendees will know in-depth about the APIs that supports them in making intelligent apps and get to see some amazing demos of what's feasible with contextual vision, and facial analysis.

Agenda

  • How to add intelligent features in your applications using Microsoft APIs
  • Demo - facial authentication and expression
  • Q&A session
